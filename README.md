# Responsive Product Gallery with Modal

## Project Overview
This project is a responsive product gallery built using HTML, CSS, and JavaScript. It displays multiple products, each with an image, title, price, and a "View Details" button. Clicking the button opens a dynamic modal showing a larger image and product description.

## Features
- Dynamic product cards created using JavaScript (no hardcoded cards).
- Responsive layout:
  - 4 cards per row on desktop
  - 2 cards per row on tablet
  - 1 card per row on mobile
- Modal popup functionality:
  - Opens with product details dynamically
  - Closes via "X" button or clicking outside
- Clean and structured folder organization.
